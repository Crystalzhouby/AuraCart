# 问题
## 欢迎词生成
修改一下欢迎词生成的位置，放到router节点处生成。
欢迎词的信息输入来源为当前用户查询和近几轮用户对话，将近几轮用户对话与当前用户查询拼接到一起发送给LLM，使LLM获取信息更加全局化，生成的欢迎词内容前后相关性上更强，
但需要额外关注当前用户查询，生成的欢迎词主要是为了应答当前用户查询的。

## 推荐理由生成
同样地，推荐理由生成也需要同时考虑当前用户查询和近几轮用户对话，但需要额外关注当前用户查询。

## 结束词生成
同样地，结束语生成也需要同时考虑当前用户查询和近几轮用户对话，但需要额外关注当前用户查询。
